# ANSSI-Audit-Light-
Mini soft developpé pour faire une pré-audit de l'appareil

⚠️ L'application doit s'executer en tant qu'administrateur ⚠️

Se logiciel accède à différent module existant de Windows (Logs, groupes, GPO, dossier windows, ressources, ect...)

<img width="600" height="398" alt="image" src="https://github.com/user-attachments/assets/b387d4ab-d288-4526-8f36-e632fde8391e" />

Je rajouterais peut-être d'autres fonctionnalités au fur et à mesure du temps. à noté qu'il peut y avoir quelques coquille ou erreurs lors du rapport. La machine n'est pas parfaite :)
